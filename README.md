# Registro de Teste de Usabilidade: Feature "Adicionar UP"

## Tela analisada
Tela de cadastro de Unidade de Produção (UP), contendo campos de localização (latitude/longitude), nome, categoria e núcleo.
![20260406-1836-37 0914049](https://github.com/user-attachments/assets/a6957fb9-784c-4875-9d15-99f0f565df11)


## 1. Identificação do Teste
- **Feature Selecionada:** Adicionar Unidade de Produção (UP)  
- **Classificação:** Tarefa  
- **Tipo de Teste:** Teste de tarefa utilizando técnica de funil para analisar a conversão do usuário em cada etapa do cadastro de uma UP.

## Objetivo do teste
Identificar dificuldades na jornada de cadastro de uma UP e entender em quais etapas do funil os usuários encontram fricção.

## Ação ou entendimento esperado
O usuário deve conseguir cadastrar uma nova UP de forma rápida e sem dúvidas, compreendendo claramente cada campo e etapa do processo. Como ele pode é esperado que ele faça isso: navegando até a tela de **Adicionar UP**, preenchendo os dados de localização e identificação e finalizando o cadastro.


## 2. Cenário e Tarefa
- **Cenário:**  
  "Você precisa cadastrar uma nova unidade no sistema. Precisa garantir que o nome e a localização sejam registrados corretamente."

- **Tarefa:**  
  Adicionar uma nova UP.


## 3. Funil de Conversão (Etapas da Tarefa)

O teste foi dividido em 4 etapas críticas para analisar onde o usuário encontra maior dificuldade:

| Etapa        | Ação do Usuário                                              | Status (Sucesso/Falha) | Observação |
|-------------|-------------------------------------------------------------|------------------------|------------|
| 1. Entrada  | Localizar "Adicionar UP" no menu e clicar                            |                        |            |
| 2. Localização | Preencher Latitude e Longitude da localização da UP  |                        |            |
| 3. Dados    | Preencher Nome, Categoria e Núcleo                          |                        |            |
| 4. Conclusão| Clicar em "Adicionar"                            |                        |            |

## Conjunto de perguntas (Funil)

1. Você conseguiu encontrar facilmente onde adicionar uma nova UP?
2. Como foi sua experiência ao inserir ou coletar a localização?
3. Os campos de dados (nome, categoria, núcleo) estavam claros para você?
4. Você teve alguma dúvida antes de clicar em "Adicionar"?
5. Você acredita que a tarefa foi concluída com sucesso? Por quê?
